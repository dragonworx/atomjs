Welcome to atomjs!

This is an example site built with atom. It demonstrates public and private content plus navigation and error handling.

You'll need to configure your server to always return the bootstrap.html page for every request. This enables front routing on the client.

Try clicking on Admin, you'll be directed to login. To login, just click the Sign In button (no credentials needed).

Try entering a url which doesn't exist, you'll see the 404 template.
